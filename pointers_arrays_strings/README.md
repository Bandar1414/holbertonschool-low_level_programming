# C - More pointers, arrays and strings

This directory contains solutions for the More pointers, arrays and
strings project.

## Files

- `0-strcat.c`: concatenates two strings.
- `1-strncat.c`: concatenates two strings using at most n bytes.
- `2-strncpy.c`: copies a string using at most n bytes.
- `3-strcmp.c`: compares two strings.
- `4-rev_array.c`: reverses an array of integers.
- `5-string_toupper.c`: changes lowercase letters to uppercase.
- `6-cap_string.c`: capitalizes words in a string.
- `7-leet.c`: encodes a string into 1337.
